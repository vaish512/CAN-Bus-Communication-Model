## 📖 Overview
This project demonstrates a basic CAN (Controller Area Network) Bus Communication Model used in automotive systems for ECU-to-ECU communication.

The simulation helps understand how automotive electronic control units exchange messages efficiently inside modern vehicles using CAN protocol.

# 🚘 Project Architecture

---

# 🚀 Steps to Run the Project

1. Open MATLAB
2. Open Simulink
3. Create a new blank model
4. Add required blocks:
   - Pulse Generator
   - Gain
   - CAN Transmit
   - CAN Receive
   - Scope
5. Connect blocks properly
6. Run simulation
7. Observe CAN communication output in Scope
